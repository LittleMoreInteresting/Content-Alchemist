# 核心定为重构
1、核心功能定义为一个公众号文章AI写作助手
2、核心功能 根据标题 + 公众号定为相关配置 生成大纲 然后根据大纲内容生成 可发布的文章内容；

# 界面重构

## 顶部菜单
1、移除 “打开” “新建” 按钮；
2、添加 “保存” 按钮；点击保存时 如果没有保存到本地文件则选择目录然后保存；默认使用 标题.md 命名文件名 已保存到本地文件则直接保存；
3、设置按钮保持不变;

## 侧边栏
1、侧边栏重构为标题输入框 + 写作要求等基本内容；
2、底部分为 “生成大纲”，“生成文章” 按钮 生成文章前必须生成大纲
3、点击生成大纲 按钮 后 根据标题 + 基本要求 发送给AI 生成大纲 大纲内容显示在页面中 用户可以对大纲内容进行修改
4、点击生成文章 按钮 后 根据大纲内容 + 基本要求 获取文章内容 显示在页面中
